칵테일 제조법

1.쉐이크(shake) :
쉐이커에 양주, 설탕, 시럽 등을 넣어 얼음과 함께 흔들어 혼합하는 것

2.싱글(single) :
술의 용량을 나타내는 것 (=30ml)

3.스퀴즈(squeeze) :
과실의 즙을 짜는 것

4.스터(stir) :
바스푼으로 술을 휘저어 섞는 것. 쉐이크할 경우 술이 탁해질 때 사용

5.스노 스타일(snow style) :
칵테일 글라스 가장자리에 레몬즙을 묻히고 그 위에 설탕을 묻혀 눈처럼 보이게 하는 것

6.슬라이스(slice) :
과일을 얇게 썬 것

7.필(peel) :
레몬이나 오렌지 조각을 강하게 움켜쥐어 칵테일에 즙을 짜 향을 내는 것
